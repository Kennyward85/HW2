ViVi/Virtual Visitor

Git Repository

This application is a portal setup allowing Doctors Offices to contact us to become a User. Once they have done this they will be given access to the sites features for setting up patients and pre-screening them for COVID symptoms. This is set up to allow the office to have the patients wait in their cars and be notified via their phones when it is their turn to be seen instead of crowded waiting rooms. Used React.js Authentication with Admin privileges and User encryption. Express, Mysql, Sequalize, react-bootstrap, Node.js, and Kanban project boards were used.

Hurricane Quiz

Git Repository

Fun quiz using Javascript, HTML, CSS. There is a countdown timer for the time to take this quiz on Hurricanes, points added for correct answers and deducted for wrong answers. Additionally time is deducted off the clock for incorrect answers. At the End the user will be able to view their high score and other high scores saved. With an option to go back and retake the quiz to try and achieve the highest score possible.
